# 🛢️ تحليل سيناريوهات الموازنة العراقية وأسعار النفط وسعر الصرف

تطبيق **Streamlit** تفاعلي يحلّل أثر **أسعار النفط** و**تخفيض سعر صرف الدينار**
على الإيرادات والعجز والاحتياطيات، مع التركيز على **الفجوة بين السعر الموازي والرسمي**
والتضخم المستورد.

🔗 **النسخة المنشورة:** https://iraqbudgetapp-vodfqp88y2obwcpjkahi6u.streamlit.app/

---

## ✨ المزايا

| التبويب | الوصف |
|---|---|
| 📊 نظرة عامة | الرصيد والإيرادات والنفقات والاحتياطي حسب سعر النفط + بطاقات مؤشرات (KPIs) وسعر التعادل |
| 💱 تخفيض سعر الصرف | أثر رفع سعر الصرف على الإيراد النفطي بالدينار والرصيد مقارنةً بالسعر الرسمي الأساس |
| ↔️ الفجوة الموازية/الرسمية | حساب الفجوة ونسبتها، وأثر سدّها جزئياً/كلياً، ومقايضة الرصيد مقابل التضخم |
| 🔥 الحساسية المزدوجة | خريطة حرارية للرصيد عند كل تركيبة (نفط × صرف) |
| 📈 التضخم والاستيراد | تقدير تقريبي للتضخم المستورد وزيادة فاتورة الاستيراد |
| 📅 البيانات الفعلية (CBI) | سلاسل رسمية من البنك المركزي (الاحتياطي، M0، التضخم، سعر الصرف، الموازنة) + **تحقق النموذج مقابل الواقع** عبر «سعر النفط الضمني» |
| 🗂️ البيانات والتصدير | الجداول الكاملة وتنزيل النتائج (Excel / CSV) |

> 🟢 **القيم الافتراضية مُعايَرة على أرقام البنك المركزي العراقي الفعلية** (Dec 2024 / Dec 2025).
> اختر سنة المعايرة من أعلى الشريط الجانبي. تحقّق النموذج: السعر الضمني المشتقّ من الإيراد
> الفعلي ≈ 82$ (2024) و71$ (2025)، وهو قريب جداً من متوسط أسعار التصدير الفعلية.

## 🧮 المنهجية (موجزة)

- **الإيراد النفطي (دينار)** = سعر النفط × صافي الصادرات الحكومية × أيام السنة × سعر الصرف
- **الرصيد** = إجمالي الإيرادات − النفقات (إجمالي أو جاري)
- **العجز** يُموَّل جزئياً من البنك المركزي ⇐ يخفض الاحتياطي
- **الفائض** يُخصَّص جزء منه للاحتياطي
- **سدّ الفجوة**: رفع السعر الرسمي نحو الموازي بنسبة قابلة للتعديل
- **التضخم المستورد** ≈ حصة السلع المستوردة × معامل التمرير × نسبة تغيّر الصرف

> ⚠️ النتائج تقديرية لأغراض تحليل السيناريوهات وليست توقعات رسمية.

## ▶️ التشغيل محلياً

```bash
python -m venv .venv
.venv\Scripts\activate            # ويندوز  (Mac/Linux: source .venv/bin/activate)
pip install -r requirements.txt
streamlit run app_ar.py
```
