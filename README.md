\# تحليل سيناريو الموازنة العراقية (عربي)



تطبيق Streamlit يحسب أثر أسعار النفط على الموازنة والاحتياطيات وفق منهجية الصفحات 12–22، مع قاعدة: نسبة من الفائض تذهب للاحتياطي.



\## تشغيل محلي

```bash

python -m venv .venv

.venv\\Scripts\\activate   # على ويندوز (Mac/Linux: source .venv/bin/activate)

pip install -r requirements.txt

streamlit run app\_ar.py





## المنهجية باختصار
- الإيرادات النفطية = (الصادرات الصافية × حصة الحكومة × السعر × 1e6 × 365 × سعر الصرف) ÷ 1e9  → **مليار دينار**.
- الرصيد = إجمالي الإيرادات − النفقات (إجمالي أو جارية حسب الاختيار).
- العجز يُموّل بنسبة من البنك المركزي ⇒ انخفاض في الاحتياطي.
- الفائض يُخصص جزء منه للاحتياطي ⇒ ارتفاع في الاحتياطي.
- **نسبة التغطية** = الاحتياطي بعد / النقد القاعدي (أخضر عندما ≥ 100%).
